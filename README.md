# 深度学习实战培训 · 暑期课程

> 面向实验室同级（研0）和师弟师妹（即将大四）的深度学习拓展培训。
> 不求多深，重在建立系统认知、动手实践、了解前沿。

## 课程主线

**计算机视觉（CV）+ 大语言模型（LLM）**，最终汇聚到**多模态**。

## 课程结构

本仓库按课时组织，每节课包含课件、讲义、代码等材料。

```

├── 第二节课/          # ...
```

## 如何开始

**零基础学员请从 `第零节课/README.md` 开始**，完成：
1. 阅读学习路线和课程计划
2. 安装 Miniconda + PyTorch（本地安装，推荐）
3. 跑通 `check_env.py`
4. 预习第一节课内容

> 其他课程库（transformers、ultralytics 等）会在**第一节课统一带大家安装**。

## 环境要求

- Python 3.11 - 3.13（推荐 3.12）
- PyTorch + CUDA（可选，CPU/MPS 也可运行）
- 课前只需装好 Miniconda + PyTorch + 最基础库
- 课程所需库的安装说明见 `第零节课/环境安装指南.md`

## 环境方案

| 方案 | 说明 |
|------|------|
| **本地安装（推荐）** | 课前装好 conda + PyTorch，第一节课统一补装其他库 |
| **Google Colab** | 无 GPU 或不想本地安装的同学备选，免费 T4 GPU |

详见 `第零节课/环境安装指南.md`。

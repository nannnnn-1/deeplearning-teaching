# 第零节课 · 课前准备

> 在正式上课之前，请先完成本节课的内容：了解课程路线、装好最基础的环境。
> 其他课程库会在**第一节课统一带大家安装**。

## 本节课目标

1. **了解全局**：知道 21 节课要学什么，12 层知识图谱长什么样。
2. **明确路线**：理解 CV + LLM 双主线，以及它们如何汇聚到多模态。
3. **配置基础环境**：装好 Miniconda + Python + PyTorch（含 CUDA/CPU/MPS），能 `import torch`。
4. **预习第一节课**：对课程内容有大致印象。

## 推荐步骤

### 第一步：看路线（10 分钟）

打开 `学习路线.md`，快速浏览 12 层知识结构。不需要记住所有内容，只需要知道：
- 深度学习有哪些方向
- 我们这 21 节课覆盖哪些方向
- 多模态在最后，但它需要 CV 和 LLM 两个基础

### 第二步：看计划（5 分钟）

打开 `课程计划.md`，了解：
- 每节课的主题
- 每节课的大致目标
- 课后作业要求

### 第三步：装基础环境（30-60 分钟）

**课前只需要完成三件事：**

1. 安装 Miniconda
2. 创建 `dl-course` 环境（Python 3.10）
3. 安装 PyTorch（带 CUDA 或 CPU 版本）

具体命令见 `环境安装指南.md`。

### 第四步：检查基础环境（2 分钟）

运行：

```bash

```

只要看到以下输出即表示课前准备完成：

```

  → CUDA 可用: ...（CPU 用户会提示使用 CPU）
✓ torchvision x.x.x
✓ matplotlib x.x.x
✓ jupyter x.x.x
```

> 其他库（transformers、ultralytics 等）第一节课会统一安装。

### 第五步：预习第一节课（10 分钟）

浏览 `../第一节课/课件.html` 和 `../第一节课/讲义.md`，对第一节课内容有大致印象。

## 常见问题

### Q1：没有 GPU 可以参加吗？

**可以。** 所有课程代码都可以在 CPU 上运行，只是速度慢一些。推荐使用 Google Colab 的免费 T4 GPU。

### Q2：课前需要把 transformers、ultralytics 这些都装好吗？

**不需要。** 第一节课会带大家边用边装，这样版本更新时讲义也不用改太频繁。

### Q3：环境安装遇到问题怎么办？

1. 仔细阅读 `环境安装指南.md` 的"常见错误排查"部分。
2. 在课程群里提问，附上完整报错信息。
3. 如果本地实在搞不定，先用 Google Colab。

### Q4：课前需要会哪些前置知识？

- Python 基础：会写循环、函数、类，会用 NumPy 做简单数组操作。
- 数学基础：高中数学 + 一点大学线代/微积分即可。课程会"用到哪讲到哪"。

# 深度学习 · 12 层知识结构

> 本培训覆盖其中 **6 层重点内容**（L3、L4、L5、L6、L7、L11），其余层在第 1 节课做全面介绍。

| 模块 | 核心内容 |
|------|----------|

| 编程与工具 | Python + NumPy、PyTorch、GPU/CUDA、Git/GitHub |

**学习策略**：不单独开课，用到哪讲到哪。

## L2：机器学习基础

| 模块 | 核心内容 |

## L3：深度学习核心（本课程重点）⭐

| 模块 | 核心内容 |
|------|----------|
| 神经网络基础 | 感知机与 MLP、反向传播、参数初始化、标准化（BatchNorm/LayerNorm） |
| 优化算法 | SGD + Momentum、Adam/AdamW、学习率调度 |

| 经典网络架构 | LeNet → AlexNet → VGG → ResNet、RNN/LSTM/GRU、Self-Attention / Transformer、Vision Transformer |

## 推荐学习路径

### CV 工程师路线
地基 → CNN（2-4节）→ ResNet（3节）→ YOLO 检测（5节）→ U-Net/SAM 分割（6-7节）→ Diffusion 生成（11-13节）

### NLP/LLM 工程师路线
地基 → Attention（8节）→ Transformer（9节）→ nanoGPT（14节）→ LoRA 微调（15节）→ SFT/DPO（16节）→ RAG/Agent（17-18节）→ 量化推理（19节）

### 多模态路线
CV 基础 + LLM 基础 → CLIP（20节）→ LLaVA/Qwen-VL（21节）

## 方案 A：本地安装（推荐）

### 1. 安装 Python 与 Miniconda

#### 下载 Miniconda

- 官网：https://docs.conda.io/en/latest/miniconda.html
- 清华镜像：https://mirrors.tuna.tsinghua.edu.cn/anaconda/miniconda/

#### 安装

**Windows**：
运行 `.exe`，勾选 **"Add to PATH"**，其余默认。

**Linux/macOS**：
```bash
bash Miniconda3-latest-Linux-x86_64.sh
# 或 macOS
bash Miniconda3-latest-MacOSX-arm64.sh
```

#### 验证

```bash
conda --version
# 应输出类似 conda 23.x.x
```

---

### 2. 创建虚拟环境

```bash
conda create -n dl-course python=3.12 -y
conda activate dl-course
```

> 所有后续安装都在 `dl-course` 环境中进行。

---

### 3. 安装 PyTorch（含 CUDA）

#### 确认 CUDA 版本

**Windows/Linux**：
```bash
nvidia-smi
# 看右上角的 CUDA Version（如 12.4）
```

**macOS**：无需 CUDA，M 系列用 MPS，Intel 用 CPU。

#### 安装命令

| CUDA 版本 | 安装命令 |
| :-------- | :------- |
| CUDA 12.x | `pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121` |
| CUDA 11.8 | `pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118` |
| CPU only | `pip install torch torchvision torchaudio` |
| macOS MPS | `pip install torch torchvision torchaudio` |

#### 验证 PyTorch + CUDA

```python
import torch
print(torch.__version__)
print(torch.cuda.is_available())      # True 表示 GPU 可用
print(torch.cuda.get_device_name(0))  # 显卡名称
```

## 常见错误排查

### 1. CUDA 版本不匹配

- 如果 `nvidia-smi` 显示 CUDA 12.4，但 PyTorch 安装的是 cu118，可能仍能运行（向下兼容），但建议版本一致。
- 重新安装对应版本的 PyTorch 即可。

### 2. `conda activate dl-course` 提示找不到命令

- Windows 用户：安装时未勾选 "Add to PATH"，可以打开 **Anaconda Prompt** 再执行。
- Linux/macOS：执行 `source ~/.bashrc` 或重启终端。

### 3. M 系列 Mac（MPS）限制

- MPS 不支持所有 PyTorch 操作，部分实验需 fallback 到 CPU：
  ```python
  if torch.backends.mps.is_available():
      device = torch.device("mps")
  else:
      device = torch.device("cpu")
  ```

### 4. `pip` 下载很慢

- 使用清华镜像：
  ```bash
  pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
  ```
